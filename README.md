# Sachet Strategy 360 - Landing Page Premium

Landing Page Premium en modo oscuro (Dark Luxury) para Sachet Strategy 360, diseñada con tecnología de neuroestrategia de empaques táctiles.

## 🎨 Paleta de Colores

| Color | HEX | RGB | Uso |
|-------|-----|-----|-----|
| Primary | `#3B82F6` | 59, 130, 246 | Acentos, botones, glows |
| Primary Light | `#60A5FA` | 96, 165, 250 | Hover states |
| Primary Dark | `#2563EB` | 37, 99, 235 | Gradientes profundos |
| BG Primary | `#000000` | - | Fondo principal |
| BG Secondary | `#09090B` | - | Fondo alternativo |
| BG Tertiary | `#18181B` | - | Cards, superficies |
| Text Primary | `#FFFFFF` | - | Títulos |
| Text Secondary | `#E4E4E7` | - | Body text |
| Text Tertiary | `#A1A1AA` | - | Labels |

## 📁 Estructura del Proyecto

```
Sachet Strategy 360/
├── index.html          # Estructura principal
├── styles.css          # Todos los estilos
├── script.js          # Animaciones e interacciones

    ├── logo.png                    # Logo (512x512)
```

## 🚀 Características

- **Dark Luxury Mode**: Diseño premium en modo oscuro
- **Glassmorphism**: Tarjetas con efecto cristal translúcido
- **Flip Cards**: Cards de servicios interactivas con hover/click
- **Particles.js**: Sistema de partículas flotantes
- **Scroll Animations**: Animaciones al hacer scroll
- **Fully Responsive**: Adaptado a todos los dispositivos
- **Plus Jakarta Sans**: Tipografía premium de Google Fonts

## 🔧 Personalización

### Cambiar Color Principal
En `styles.css`, busca y reemplaza el color:
```css
--primary-500: #3B82F6; /* Tu color nuevo */
```

### Cambiar Imágenes
Simplemente reemplaza los archivos en la carpeta `images/` con tus propias imágenes.
