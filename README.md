## 🚀 Características

- **Dark Luxury Mode**: Diseño premium en modo oscuro
- **Glassmorphism**: Tarjetas con efecto cristal translúcido
- **Flip Cards**: Cards de servicios interactivas con hover/click
- **Particles.js**: Sistema de partículas flotantes
- **Scroll Animations**: Animaciones al hacer scroll
- **Fully Responsive**: Adaptado a todos los dispositivos
- **Plus Jakarta Sans**: Tipografía premium de Google Fonts

## 🔧 Personalización

### Cambiar Color Principal
En `styles.css`, busca y reemplaza el color:
```css
--primary-500: #3B82F6; /* Tu color nuevo */
```

### Cambiar Imágenes
Simplemente reemplaza los archivos en la carpeta `images/` con tus propias imágenes.
